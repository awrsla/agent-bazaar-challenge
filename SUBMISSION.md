## Accepted Public Links

Use one or more of:

- public repository
- public cast
- public post
- demo video
- hosted bot/tool
- command output with explanation
- market report generated by the agent

## Categories

Choose the closest category:

1. Bazaar signal bot
2. Ingredient market report
3. Uncharted cargo planner
4. Leaderboard watcher
5. Human vs agent market commentary
6. Best use of Chef-Universe/skills
7. Other useful Chef Universe agent

## Minimum Qualification

A submission must:

- use real Chef Universe data
- produce useful output
- include a wallet address
- avoid guaranteed-profit claims
- avoid spam or copied submissions

## Review

Submissions are reviewed after the campaign window.

Qualified participant rewards are capped at the first 10 valid submissions and require review. Top agents are selected using the judging criteria in README.md.
